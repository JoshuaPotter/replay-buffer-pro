What is Replay Buffer Pro?

Replay Buffer Pro is a plugin for OBS Studio that expands upon built-in Replay Buffer. When you press a button or hotkey for quick-save preset, the plugin saves the current buffer and instantly trims it down to the desired length, without re-encoding.

Key Features

Quick-save presets

Save the last 15s, 30s, 1m, 5m, 15m, or 30m instantly.

Dockable, minimal UI

Controls are always within reach.

Instant clips

No re-encoding, same quality as your buffer.

Hotkeys

Assign hotkeys for each preset length in settings.

Installation

Download for Windows (x64)

Requirements

  • OBS Studio 30.0.0+
  • Windows 10/11 (64-bit)
  • macOS and Linux are not currently supported (coming soon!)

Instructions

  1. Download the latest release of Replay Buffer Pro.
  2. Close OBS Studio.
  3. Extract the ZIP and merge the obs-studio folder into your OBS Studio installation directory (typically C:\Program Files\obs-studio\).
  4. Verify the final installation structure resembles:
obs-studio/
├── obs-plugins/
│   └── 64bit/
│       └── replay-buffer-pro.dll
└── data/
    └── obs-plugins/
        └── replay-buffer-pro/
            ├── locale/
            │   └── en-US.ini
            └── ffmpeg.exe

Tip: For Windows, you may need administrator privileges to copy files into Program Files.

Getting Started

Usage

  1. Open OBS and start the Replay Buffer.
  2. Open the Replay Buffer Pro dock (View → Docks → Replay Buffer Pro).
  3. Click a save button (e.g., “Last 30 Seconds”) or trigger its hotkey.
  4. To change how much OBS stores, stop the Replay Buffer, then adjust the buffer length slider or seconds field.

Tips

  • Hotkeys: Assign keys for each duration in OBS Studio Settings → Hotkeys (search “Replay Buffer Pro”).
  • Disabled buttons: A clip button is disabled if your buffer isn't long enough (e.g., 5m clip requires a ≥5m replay buffer).
  • Adjusting buffer length: Use the slider in the plugin UI; no need to dig through settings.
  • Where clips save: They're written to the same folder OBS Studio uses for Replay Buffer saves (as configured in your OBS Studio Output settings).

FAQ

How is this different from OBS's default Replay Buffer?

The default saves a single fixed duration clip. Replay Buffer Pro lets you save multiple preset lengths on demand with buttons/hotkeys, and trims automatically.

Why would I use this?

Use Replay Buffer Pro if you want to save recent moments of your stream or recording at different lengths, without manually clipping them from your final recording or stream. This plugin seeks to recreate Xbox & Playstation's "Save Recent Gameplay" functionality. Perfect for preparing clips for TikTok, YouTube Shorts, and Instagram Reels.

Does it re-encode video?

No. Trimming uses FFmpeg with -c copy for fast, lossless output.

Why are some save buttons disabled?

Your configured buffer length is shorter than that clip length. Increase the buffer length or choose a shorter clip.

Why can't I change the buffer length right now?

The Replay Buffer is active. Stop it to adjust the length, then start it again.

Can I change the preset durations?

Not currently. The UI provides fixed presets.

Where are clips saved?

In your OBS Studio Replay Buffer save directory (set in OBS Studio Output settings).

Troubleshooting

  • Ensure replay-buffer-pro.dll is in obs-studio/obs-plugins/64bit/.
  • If trimming fails, confirm ffmpeg.exe exists at obs-studio/data/obs-plugins/replay-buffer-pro/.
  • Check OBS Studio log for plugin load messages and errors.
  • Close OBS Studio before installing or updating the DLL; use an elevated terminal if copying into Program Files.
