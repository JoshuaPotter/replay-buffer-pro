What is Replay Buffer Pro?

Replay Buffer Pro is a plugin for OBS Studio that expands upon the built-in Replay Buffer. When you press a button or hotkey for quick-save preset, the plugin saves the current buffer and instantly trims it down to the desired length, without re-encoding.

Key Features

Quick-save presets

Save the last 15s, 30s, 1m, 5m, 15m, or 30m instantly.

Instant clips

No re-encoding, same quality as your buffer.

Hotkeys

Assign hotkeys for each preset length in settings.

Dockable, minimal UI

Simple user interface designed for quick access.

FAQ

How is this different from OBS's default Replay Buffer?

The default saves a single fixed duration clip. Replay Buffer Pro lets you save multiple preset lengths on demand with buttons/hotkeys, and trims automatically.

Why would I use this?

Replay Buffer Pro is ideal for effortlessly capturing and saving recent moments from your streams or recordings in various lengths. Unlike traditional methods that require manual clipping, this plugin mimics the "Save Recent Gameplay" feature found on Xbox and Playstation, ensuring seamless and automatic trim functionality. It's the perfect tool for crafting clips ready for platforms like TikTok, YouTube Shorts, and Instagram Reels.

Does it re-encode video?

No. Trimming uses FFmpeg with -c copy for fast, lossless output.

Why are some save buttons disabled?

Your configured buffer length is shorter than that clip length. Increase the buffer length or choose a shorter clip.

Why can't I change the buffer length right now?

The Replay Buffer is active. Stop it to adjust the length, then start it again.

Can I change the preset durations?

Not currently. The UI provides fixed presets.

Where are clips saved?

In your OBS Studio Replay Buffer save directory (set in OBS Studio Output settings).

Troubleshooting

Plugin Not Loading

Ensure replay-buffer-pro.dll is in obs-studio/obs-plugins/64bit/. Restart OBS Studio after placing the file.

Trimming Fails

Confirm ffmpeg.exe exists at obs-studio/data/obs-plugins/replay-buffer-pro/. The plugin requires FFmpeg for clip processing.

Check the Logs

Review the OBS Studio log file for plugin load messages and errors. Look for "Replay Buffer Pro" entries to diagnose issues.

Report an Issue

If you're experiencing a bug or need additional help, please file an issue on GitHub. Include your OBS Studio version, plugin version, and any relevant log entries.
